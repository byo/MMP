Look at this short documentation: https://github.com/idler/MMP/wiki/Description.

To install MMP, add this to your composer.json:
```
    "require": {
        "mmp/mmp": "dev-master@dev"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/idler/MMP"
        }
    ]
```
